Deploy to Cloudflare Pages

Deploy your site globally with Cloudflare's fast and secure network.

Step 1: Create Application

Log in to the Cloudflare Dashboard. Click + Add and select Pages.

Then select Connect to Git.

Step 2: Select Repository

Authorize Cloudflare to access your GitHub account and select your repository (e.g., it464a-gitsample).

Step 3: Configure Builds

Set the Project Name (this will be your subdomain). For Framework preset, leave it as "None" for a static HTML site.

Step 4: Deploy

Click Save and Deploy. Cloudflare will build and deploy your site in seconds.
